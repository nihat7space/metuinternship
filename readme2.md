# ✈️ Airline Reservation System

## 📝 Description
A simple airline seat booking system with:
- Economy/Business class selection
- Capacity management
- Investment simulation

## 🛠 Technologies
- **Language**: C
- **Concepts**: Function pointers, basic I/O, recursive investment calculation
